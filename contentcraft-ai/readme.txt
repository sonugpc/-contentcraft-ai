AI-powered content enhancement and generation for WordPress using Google's Gemini API.

== Description ==

ContentCraft AI is a powerful WordPress plugin that leverages Google's Gemini AI to enhance your content creation workflow. Whether you're looking to improve existing content or generate new posts from scratch, ContentCraft AI provides intelligent assistance right within your WordPress editor.

= Key Features =

* **Content Enhancement**: Improve existing content with AI-powered suggestions
* **Content Generation**: Create new content from titles and topics
* **Gutenberg Integration**: Seamlessly works with the block editor
* **Classic Editor Support**: Full compatibility with the classic editor
* **Customizable Prompts**: Tailor AI responses to your specific needs
* **Rate Limiting**: Built-in controls to manage API usage
* **Secure API Key Storage**: Encrypted storage of your Gemini API key
* **Usage Tracking**: Monitor your API usage and stay within limits

= How It Works =

1. **Get your Gemini API Key**: Sign up at Google AI Studio and get your API key
2. **Configure the Plugin**: Enter your API key and customize prompt templates
3. **Start Creating**: Use the AI tools directly in your post editor
4. **Enhance or Generate**: Choose to enhance existing content or generate new content
5. **Review and Accept**: Review AI suggestions and accept the ones you like

= Editor Integration =

ContentCraft AI integrates seamlessly with both Gutenberg and Classic editors:

* **Gutenberg**: Adds a sidebar panel and floating button for easy access
* **Classic Editor**: Adds a button to the editor toolbar
* **Modal Interface**: Clean, user-friendly interface for all AI operations

= Customization =

The plugin offers extensive customization options:

* **Prompt Templates**: Create custom prompts for different content types
* **API Settings**: Configure token limits, temperature, and other AI parameters
* **Rate Limiting**: Set hourly usage limits to control costs
* **Logging**: Enable detailed logging for debugging and monitoring

= Security & Privacy =

* **Encrypted API Keys**: Your API key is stored securely using WordPress encryption
* **Rate Limiting**: Prevents API abuse and unexpected charges
* **Content Sanitization**: All AI-generated content is properly sanitized
* **Permission Checks**: Ensures only authorized users can access AI features

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/contentcraft-ai` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->ContentCraft AI screen to configure the plugin
4. Get your Gemini API key from Google AI Studio
5. Enter your API key and customize your settings
6. Start using the AI features in your post editor!

== Frequently Asked Questions ==

= Where do I get a Gemini API key? =

You can get a free Gemini API key from Google AI Studio at https://makersuite.google.com/app/apikey

= Is there a free tier for the Gemini API? =

Yes, Google provides a generous free tier for the Gemini API. Check Google's current pricing for details.

= Can I customize the AI prompts? =

Absolutely! The plugin includes customizable prompt templates that you can modify to suit your specific needs.

= Does this work with page builders? =

The plugin is designed to work with WordPress editors. Integration with popular page builders may be added in future versions.

= Is my API key secure? =

Yes, your API key is encrypted and stored securely using WordPress's built-in encryption functions.

= Can I limit API usage? =

Yes, the plugin includes rate limiting features to help you control API usage and costs.
