AI-powered content enhancement and generation for WordPress using the Gemini API.

== Description ==

ContentCraft AI is a powerful plugin that integrates the Google Gemini API directly into your WordPress editor. It allows you to enhance your existing content and generate new content with the power of AI.

Features:
*   **Enhance Content:** Improve the quality of your existing content with a single click.
*   **Generate Content:** Generate new content from a title and tags.
*   **Custom Prompts:** Customize the prompts used for enhancement and generation to get the perfect results.
*   **Structured Data:** The AI returns structured data, including an enhanced title, content, tags, meta description, and focus keyword.
*   **Rank Math Integration:** The plugin automatically populates the meta description and focus keyword fields in the Rank Math SEO plugin.

== Installation ==

1.  Upload the `contentcraft-ai` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to 'Settings' > 'ContentCraft AI' to configure the plugin.
